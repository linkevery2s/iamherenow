## 今ここ。- 位置情報共有アプリ

![top](https://github.com/user-attachments/assets/736f86ed-9d95-4750-a80c-89e3e67bac2b)

このアプリは、自分の居場所を地図上で把握し「今ここ。」と伝えるアプリです。災害が発生した時の安否確認、犯罪目撃情報や災害発生箇所の位置情報の共有、友人との待ち合わせ、など、あらゆる場面で利用できます。
